# 🍽️ Time to Eat

A stylish, responsive food menu website that showcases popular dishes and desserts using modern HTML and CSS.

---

## 🚀 Features

- 🧾 Interactive and responsive menu cards
- 🍔 Showcases a variety of dishes (burgers, pasta, biryani, desserts, etc.)
- 🎨 Custom design using `Poppins` font and clean layout
- 🌐 Mobile-friendly UI
- 📷 Image-rich food display for visual appeal

---

## 📂 Tech Stack

- HTML5
- CSS3
- Google Fonts
- Custom responsive styling

---

## 🎯 Purpose

This project was built to practice and showcase front-end web development skills. It serves as a portfolio piece that demonstrates:

- Creative UI design
- Responsive layout structuring
- Use of semantic HTML
